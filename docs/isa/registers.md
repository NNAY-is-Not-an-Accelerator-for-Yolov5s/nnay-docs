# 寄存器定义

在本文档中，指代名为`name`的寄存器时，统一使用`Reg(name)`表示。

## 配置寄存器

### MX 模式寄存器

| 名称          | 位宽 | 描述       |
| ------------- | ---- | ---------- |
| MXMode        | 4    | - |

MXMode 的值域为：

- `0b0000`: 不使用MX模式
- `0b0001`: 使用MX9模式

其余值域保留。

### 基地址寄存器

#### 输入基地址寄存器

| 名称          | 位宽 | 描述       |
| ------------- | ---- | ---------- |
| InputBaseAddr | 5    | 输入基地址 |

#### 输出基地址寄存器

| 名称           | 位宽 | 描述       |
| -------------- | ---- | ---------- |
| OutputBaseAddr | 5    | 输出基地址 |

#### 权重基地址寄存器

| 名称          | 位宽 | 描述       |
| ------------- | ---- | ---------- |
| WeightBaseAddr | 5    | 权重基地址 |
