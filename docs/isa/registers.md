# 寄存器定义

在本文档中，指代名为`name`的寄存器时，统一使用`Reg(name)`表示。

## 配置寄存器

### 基地址寄存器

#### 输入基地址寄存器1

| 名称          | 位宽 | 描述       |
| ------------- | ---- | ---------- |
| InputBaseAddr.1 | 5    | 输入基地址 |

#### 输入基地址寄存器2

| 名称          | 位宽 | 描述       |
| ------------- | ---- | ---------- |
| InputBaseAddr.2 | 5    | 输入基地址 |

#### 输出基地址寄存器1

| 名称           | 位宽 | 描述       |
| -------------- | ---- | ---------- |
| OutputBaseAddr.1 | 5    | 输出基地址 |

#### 输出基地址寄存器2

| 名称           | 位宽 | 描述       |
| -------------- | ---- | ---------- |
| OutputBaseAddr.2 | 5    | 输出基地址 |

#### 权重基地址寄存器

| 名称          | 位宽 | 描述       |
| ------------- | ---- | ---------- |
| WeightBaseAddr | 5    | 权重基地址 |
